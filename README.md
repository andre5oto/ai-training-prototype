# Prototype Design for AI Leadership Training UI

Here's your **AI Skills Leadership Trainer** prototype — a fully interactive, BetterUp-style learning platform ([goto prototype page](https://andre5oto.github.io/ai-training-prototype/)).

### High-level Feature Requirements:

* **Dashboard** with a personalized greeting, 12-day streak chip, and 3 stat cards (modules completed, AI fluency score, team rank)
* **6 complete learning modules** covering Prompt Engineering, Context & Role-Based Prompting, Critical Evaluation, AI Safety, Workflows, and AI Strategy — each with real progress bars
* **Interactive lesson modal** — clicking any module opens a 4-step lesson with content slides, formatted prompt examples, tip boxes, and a graded quiz at the end
* * **Sidebar nav** with cohort branding, active states, and user profile chip — just like BetterUp's left rail
* **Activity feed** and **cohort leaderboard** in the bottom section, with the user's own rank highlighted
